▶ Start notes

House rules

This repo is not trying to be a giant game project. It is for small playable prototypes that can quickly test feel, mood, and tiny systems.

1

Playable before perfect

Build the playable loop first, then grow the art, levels, and pacing. Fun gets to arrive before polish.

2

One tab, zero ceremony

Static HTML comes first: no install, no account, no backend. Open the link and start playing.

3

Small systems, big feel

Use tiny interaction systems to create character, feedback, and rhythm. The engineering can stay light; the feel should not.
